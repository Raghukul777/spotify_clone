# 🎵 Spotify Clone

A Spotify-inspired music player UI built using **React**, **Vite**, **Tailwind CSS**, and **Context API**.  
This project focuses on recreating the core UI and basic playback functionality of Spotify.

---

## 🚀 Features

- 🎧 Play / Pause songs  
- ⏭ Next & Previous track navigation  
- 📀 Album and song listing  
- 🟢 Interactive seek bar (UI)  
- 📱 Fully responsive design  
- ⚛️ Global state management using Context API  

---

## 🛠 Tech Stack

- **React** – UI library  
- **Vite** – Fast build tool  
- **Tailwind CSS** – Utility-first styling  
- **Context API** – State management
